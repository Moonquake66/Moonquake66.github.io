linux基础命令及shell使用

1.文件和目录操作命令

1.1 ls命令

用于列出目录的内容，包括文件和子目录。

1.2 cd命令（用于切换当前工作目录）

1.3 pwd命令

打印当前工作目录的绝对路径。

1.4 mkdir命令

用于创建新的目录。

1.5 rmdir命令

用于删除空目录。

1.6 rm命令

用于删除文件或目录。

1.7 CP命令

用于复制文件和目录。

1.8 mv命令

用于移动或重命名文件和目录。

2.文件查看和编辑命令

2.1 cat命令

用于查看文件内容、连接文件并输出到标准输出。

2.2 less命令

用于分页查看文件内容，比cat命令更灵活，适合查看大型文件。

2.3 head命令

用于查看文件的开头部分内容。

2.4 tail命令

用于查看文件的结尾部分内容。

3.系统信息和进程管理命令

3.1 uname命令

用于打印系统信息。

-a：打印所有系统信息，包括内核名称、主机名、内核版本、硬件平台等。

3.2 top命令

动态显示系统中各个进程的资源占用情况，如 CPU 使用率、内存使用率等。

按下 “q” 键退出top命令。

3.3 ps命令

用于查看当前系统中的进程状态。

-ef：显示所有进程的详细信息，包括进程 ID（PID）、父进程 ID（PPID）、用户、CPU 使用率等。

3.4 kill命令

用于终止进程。
